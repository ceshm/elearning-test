# e-learning

e-learning test backend

Developed using:
- virtualenv with python 3.6.x
- Django==3.0.3
- Sqlite3 db for convenience

## Project setup


#### On an active env with python>=3.6:

1. Install requirements
```console
$ pip install -r requirements.txt
```

2. Run migrations
```console
$ python manage.py migrate
```
